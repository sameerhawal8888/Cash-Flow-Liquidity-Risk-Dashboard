# Cash Flow & Liquidity Risk Dashboard

## 📌 Overview
Real-time liquidity monitoring solution tracking 
core financial health indicators across multiple 
reporting periods for finance teams.

## 🛠️ Tools Used
- Power BI (Advanced DAX)
- Excel
- Finance & Risk Analytics

## 📊 Key Features
- Operating cash flow monitoring
- Working capital tracking
- Current ratio analysis
- Automated DAX risk flag system
- Cash stress condition detection

## 🔍 Key DAX Measures
- Current Ratio (threshold-based)
- Operating Cash Flow trend
- Working Capital movement
- Risk Flag DAX logic

## 💼 Business Impact
DAX-driven risk flag reduced manual risk review 
cycles by 35% by automatically identifying cash 
stress when current ratio dropped below threshold.
